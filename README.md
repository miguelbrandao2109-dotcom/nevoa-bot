👻 Névoa Bot

A Névoa Bot é uma poderosa bot fantasma desenvolvida para auxiliar na administração, proteção e automação de grupos. Seu objetivo é manter a comunidade organizada, segura e divertida através de sistemas inteligentes de moderação e ferramentas avançadas.

---

✨ Principais Recursos

🛡️ Moderação Automática

- Anti-spam
- Anti-flood
- Anti-links
- Anti-trava
- Anti-fake
- Anti-invasão
- Sistema de advertências

👥 Gerenciamento de Membros

- Boas-vindas automáticas
- Mensagem de despedida
- Controle de administradores
- Sistema de níveis
- Registro de atividades

⚙️ Automação

- Respostas automáticas
- Comandos personalizados
- Mensagens programadas
- Sistema de tags
- Atalhos rápidos

🎮 Diversão

- Jogos interativos
- Sistema de ranking
- Perfil de usuário
- Economia virtual
- Recompensas por atividade

🔒 Segurança

- Monitoramento 24/7
- Proteção contra ataques
- Logs completos
- Backup de configurações
- Controle avançado de permissões

📊 Estatísticas

- Total de mensagens
- Usuários ativos
- Relatórios automáticos
- Dados do grupo em tempo real

---

🚀 Instalação

git clone https://github.com/seuusuario/nevoa-bot
cd nevoa-bot
npm install

Inicie a bot:

npm start

---

📌 Comandos Principais

Administração

!ban
!kick
!mute
!unmute
!warn
!unwarn
!clear

Configuração

!antilink on
!antilink off
!antispam on
!antispam off
!welcome on
!welcome off

Utilidades
